如果你有**营业收入**(独资生意、自由职业、佣金、网店、合伙人分润),你报的是 **B 表(Borang B)**,不是打工人用的 BE 表。这篇带你从头到尾搞清楚怎么报。

## 谁需要报 B 表?

只要你在该课税年度有以下任一种收入,就属于 B 表:

- 独资生意 / enterprise 的利润
- 自由职业、专业服务(设计、咨询、医生、律师等)
- 佣金 / 抽成收入(保险、地产中介等)
- 网店 / 线上卖家、gig 经济
- 合伙生意的分润

> 只有**就业收入(薪水)**、没有营业收入的人报 **BE 表**。一旦有营业收入,即使你同时也打工,也要报 **B 表**(薪水填在里面的"就业收入"栏)。

## B 表 vs BE 表

| | B 表 | BE 表 |
|---|---|---|
| 适用 | 有营业收入的居民个人 | 只有就业收入的居民个人 |
| 纸本截止 | 6月30日 | 4月30日 |
| **e-Filing 截止** | **7月15日** | 5月15日 |
| 税率 | 个人累进 0%–30% | 个人累进 0%–30% |

非居民个人用的是 **M 表**;公司(Sdn Bhd)用 **C 表**。

## 报之前先准备

- 生意的**损益表**(营业额、成本、开支)
- 设备/机器/车辆清单(算**资本减免**用)
- EA 表(如果你也有受雇收入)
- 各项减免单据(EPF、保险、生活方式、医疗、教育等)
- 银行账号(退税用)

## e-Filing 填 e-B 步骤

1. 登入 **MyTax** 门户(mytax.hasil.gov.my)
2. 选 **e-Filing → e-B**,选对课税年度(YA)
3. 填**营业收入**:营业额 − 可扣开支 − 资本减免 = 调整后营业收入
4. 填**其他收入**:就业、租金、利息等
5. 填**减免与回扣**:个人 RM9,000、EPF、保险、生活方式、子女、父母医疗等
6. 系统自动按累进税率算出应缴税
7. **提交**,然后在截止日前**缴清余税**

算法一句话:

```
营业额 − 可扣开支 − 资本减免 = 调整后营业收入
+ 其他收入 − 个人减免 = 应税收入 → 累进税率(0%–30%)
```

## 最常见的 5 个错误

1. **把私人/家庭开支当营业开支扣** — 只有"完全且专为"赚取收入的开支可扣。
2. **用会计折旧** — 折旧不能扣,要改用**资本减免**(用不完可结转)。
3. **漏掉 CP500 分期** — 有营业收入的人通常会收到 CP500 分期缴税单,别忽略。
4. **过门槛没注册 SST** — 应税营业额过门槛要注册。
5. **迟报** — 迟交 B 表会按所得税法令被罚款,别拖到最后一天。
